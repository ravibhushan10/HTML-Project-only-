HTML Tags in Depth

This page explains commonly used HTML tags with examples.

1. Document Structure Tags

<!DOCTYPE html> — defines the document type (HTML5).
<html> — root element of an HTML page.
<head> — contains metadata like title, charset, scripts.
<title> — sets the title shown in browser tab.
<body> — contains the visible page content.

2. Text Formatting Tags

<h1> to <h6> — headings, h1 is largest and h6 is smallest.

Importance:The way of writing of heading tags decides the SEO and importance of content so you follow the proper ordering writing the heading tags.


<p> — defines a paragraph of text.

<br> — line break, moves text to the next line.

<hr> — horizontal rule, creates a line break with a divider.

<b> or <strong> — bold text, strong has semantic meaning.

<i> or <em> — italic text, em adds emphasis.

<u> — underlined text.

<sup> — superscript (like x2).

<sub> — subscript (like H2O).

3. Links and Media

<a href="URL"> — hyperlink, e.g. Google

<img src="path" alt="text"> — image element.

<audio> and <video> — embed audio and video content.

4. Semantic Tags

<header> — page or section header.
<nav> — navigation menu.
<article> — self-contained article.
<section> — thematic grouping of content.
<aside> — side content (sidebar).
<footer> — page or section footer.

5. Miscellaneous Tags

<span> — inline container for text.
<div> — block container for grouping elements.
<mark> — highlight text.
<code> — represent code snippets.
<pre> — preformatted text (preserves spaces/line breaks).
