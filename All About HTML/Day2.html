HTML Lists and Tables in Depth

This page explains all about lists and tables with detailed examples.

1. Lists in HTML

Lists are used to group related items. HTML provides three main types of lists.

1.1 Unordered List (<ul>)

Items appear with bullets by default.

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>
  • Apple
  • Banana
  • Cherry

1.2 Ordered List (<ol>)

Items appear with numbers or letters.

<ol type="A" start="3">
  <li>First</li>
  <li>Second</li>
</ol>
  1. First
  2. Second

Attributes of <ol>:

  • type: defines numbering style (1, A, a, I, i)
  • start: defines the starting number
  • reversed: reverses the order

1.3 Nested Lists

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables</li>
</ul>
  • Fruits
    • Apple
    • Banana
  • Vegetables

1.4 Description List (<dl>)

Used for terms and descriptions.

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

2. Tables in HTML

Tables display data in rows and columns. Each table consists of table rows (<tr>) containing header cells (<th>) or data cells (<td>).

2.1 Basic Table

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ravi</td>
    <td>22</td>
  </tr>
</table>
Name Age
Ravi 22

2.2 Table with Caption, Head, Body, and Foot

<table border="1">
  <caption>Monthly Sales</caption>
  <thead>
    <tr>
      <th>Month</th>
      <th>Sales</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>January</td>
      <td>100</td>
    </tr>
    <tr>
      <td>February</td>
      <td>150</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>250</td>
    </tr>
  </tfoot>
</table>
Monthly Sales
Month Sales
January 100
February 150
Total 250

2.3 Merged Cells (colspan and rowspan)

<table border="1">
  <tr>
    <th>Item</th>
    <th>Quantity</th>
    <th>Price</th>
  </tr>
  <tr>
    <td rowspan="2">Fruit</td>
    <td>Apple</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Banana</td>
    <td>30</td>
  </tr>
  <tr>
    <td colspan="2">Total</td>
    <td>80</td>
  </tr>
</table>
Item Quantity Price
Fruit Apple 50
Banana 30
Total 80

2.4 Scope Attribute

Scope helps screen readers understand the relation between headers and cells.

<table border="1">
  <tr>
    <th scope="col">Name</th>
    <th scope="col">Age</th>
  </tr>
  <tr>
    <th scope="row">Ravi</th>
    <td>22</td>
  </tr>
</table>
Name Age
Ravi 22
