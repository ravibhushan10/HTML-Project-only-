HTML Forms - In Depth

HTML forms are used to collect input from users. A form contains different input elements such as text fields, checkboxes, radio buttons, file selectors, and submit buttons. Forms send data to a server for processing using attributes like action and method.

Basic Form Structure

<form action="submit.php" method="post">
   ...form elements...
</form>

action specifies the URL where data is sent. method can be GET (data in URL) or POST (data hidden in body).

Form Attributes

  • action: URL to send data.
  • method: GET or POST.
  • name: Name of the form.
  • target: Where to display response (_blank, _self, etc.).
  • enctype: Encoding type for file upload (e.g., multipart/form-data).
  • autocomplete: On/Off for autofill.
  • novalidate: Disable validation.
